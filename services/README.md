
# Services

This directory contains Docker services for local development only.

⚠️ **Warning**: The `docker-compose.yml` file in this directory is configured for **local builds** and development purposes only. It should not be used in production environments.

## Setup

```bash
docker-compose up
```

## Notes

- Configuration and credentials are set for local development
- Not optimized or secured for production use
- For production deployments, create appropriate configurations following your organization's standards
